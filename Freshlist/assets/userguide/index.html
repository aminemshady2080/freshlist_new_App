Welcome To Freshlist App

This is a user Guidance or Help of How to Use this App which help you make a list of activities you need use to do for you.

How To Login

When launched this app for first time you will be asked to login, so provide you email to login or You can click on sign Up button so you can signup


How To SignUp

If you are a new user (didnt have account before) from login page you can click on sign up button, so you will be presented with a for to fill in with your username,email, password and repeat password. Provide them and click a button below labeled Sign Up.

After that pay attention to see the message we give you.if The message says thank to SignUp,

then you can check in your email inbox or spams you will have an activation email,click on it to activate your account.Then you can come back to freshlist on login with your email.Once you logged in you will stay logged in until you log out yourself .

How To Log Out

If you need to log out you can log out yourself from app when you click on 3 dots or menu >> More in the dropdown list click on Log Out.


User Dashboard

User Dashboard is like a home page for the user where you have all the list you have made grouped according services we provide. User dashboard has Services cathegories which contains list inside or empty if no lists.
Click on any service name to expands all lists you made in that service cathegory.

To Create a New List

To create a new list you need to click on bottom button lebel add(+) and you will need to give the list a name (any name you want), you must click on that text to schedule the time when this list should be executed by us.

once you have confirmed to set time scheduled time you will see to choices if this list belongs to which service,check one of them and confirm the list details, that's all;



To Add Items on a List

After building a list you can add items to a list by giving each item a name but before adding it to list you can provide more information as note in the dialog which will be presented to you after clicking on add(+) button or enter key from your keyboard.


After adding all items you need then you can send that list by clicking on send button.

To Add Your Address

To add your address or an address where will deliver our services for you, you need give an address and more information about it as shown below.



To Send Order

To send order you just need have filled all you must fill and check if the list you made is correct then click on send then you will see a message telling your order status. if you dont have internet connection turned on you will be presented with 2 option click one to turn on internet connection and then resend the order.

To Use and exiting List

You can use an existing list( a list that you have made before not yet deleted). To use one of those list go to dashboard, expand the services list click on the name of the list you need,then you can add new items to exiting ones you can rename it by clicking on list details on botton center button and schedule this list to a new schedule time. Or you can delete this list by clicking on delete button on bottom right button.Or Just Send the list using Left bottom button send.



To Change User Settings

To Change user settings you need to click any where in the Upper area of Dashboard screen or on settings icon on top screen, click on each settings you need to change, you will need current password to be able to change anything.


once you provided new value then click Update.

To Change Profile picture

You can change the profile picture from settings screen click on profile picture from top of list,you have 2 options:
  1. Picture from gallery: choose a small sized picture
  2. Take a new picture: using the camera
once done you can see the profile picture on dashboard screen,go back to see how it looks.
